# RizzCapture - AI Resilience Architecture

A static reference website for the Jefferson Cloud Marketing Methodology, demonstrating the "Trotlines, Nets, and Lures at Different Depths" approach to AI resilience architecture marketing.

## Overview

RizzCapture serves as a factual, de-identified reference site that maps business expenses (networking, travel, R&D) to standardized marketing frameworks using a fishing metaphor.

### The Methodology

1. **Trotlines (The Long Game):** Automated, low-level scanning tools that run in the background
2. **Nets (The Forecast):** Predictive analytics that catch market shifts before they arrive  
3. **Lures (Social Depth):** The 'Socialize, Decode, Consult' funnel across different engagement depths

## Framework Reference Pages

The site includes comprehensive reference pages for each marketing framework:

### Input Stage (Risk Radar)
- **[PESTEL Analysis](pestel.html)** - Political, Economic, Social, Technological, Environmental, Legal factors
- **[Porter's Five Forces](porters-five-forces.html)** - Competitive landscape analysis

### Analysis Stage (Responsibility Futures)  
- **[Bourdieu's Distinction](bourdieu-distinction.html)** - Symbolic capital intelligence framework

### Strategy Stage (Growth)
- **[Ansoff Matrix](ansoff-matrix.html)** - Market penetration, development, and diversification strategies

### Action Stage (Campaign Canvas)
- **[7Ps Marketing Mix](7ps-marketing-mix.html)** - Product, Price, Place, Promotion, People, Process, Physical Evidence

## Deployment

This site is designed for GitHub Pages deployment:

1. Enable GitHub Pages in repository settings
2. Select "Deploy from a branch" 
3. Choose "main" branch and "/ (root)" folder
4. Update `_config.yml` with your GitHub Pages URL
